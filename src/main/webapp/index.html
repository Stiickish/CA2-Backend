About this project

This project is based on the 3rd semester startcode, made by the group Koksikoden, for educational and private use. This project is a 3rd semester project, that showcase the use of different tools as Java,Tomcat, MYSQL etc.


How this project is build: This project includes a security section using JWT tokens for validations, different uses of DTO's, threads and data fetching from api's on the web. The whole idea of this project is to show, how to use different tools in backend, like fetching data from different APi's and how to combine them into one DTO and send that info further on. The project was ready to use when cloned into, and only minor changes has to be made.
